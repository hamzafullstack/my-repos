block and inline

most of the element we have learned about so far are block element. display: block

inline elements do not start from new line

div and span

Sometimes, you just need a container you can style or position! That’s where div and span come in.

A div is a generic block-level container. best use case grouping cards, sections, sidebars, or navigation bars.

A span is a generic inline container.

div participates in the flow as a block-level element, while span behaves as an inline element.

if you want to control the display property of an inline element use CSS display Block or inline block
