Inspecting HTML and CSS

Developer Tools (DevTools) allow you to inspect and test the HTML and CSS of any webpage directly in your browser. It is one of the most useful tools for debugging websites.

Open DevTools

Method 1

  • Press F12

Method 2

  • Right-click anywhere on the webpage.
  • Click Inspect.

Main Panels

1. Elements Panel

Displays the HTML structure of the webpage.

  • View HTML elements
  • Expand and collapse tags
  • Select any element
<body>
    <h1>Hello World</h1>
    <p>Welcome!</p>
</body>

2. Styles Panel

Displays the CSS applied to the selected element.

  • View CSS rules
  • Edit CSS live
  • Add new CSS properties
  • Disable CSS temporarily

Selecting an Element

  • Click the Select Element icon.
  • Hover over any webpage element.
  • Click the element.
  • DevTools highlights its HTML and CSS.

Strikethrough CSS

color: red;
color: blue;

The browser will apply: 

color: blue;
Meaning:
The crossed-out CSS property has been overridden by another rule and is not currently applied.

Live CSS Editing

Before:

font-size: 16px;

Change to:

font-size: 30px;

The webpage updates instantly.

Changes made in DevTools are temporary. Refreshing the page restores the original CSS.

Why Use DevTools?

  • Debug HTML
  • Debug CSS
  • Test new styles
  • Fix layout issues
  • Inspect any website
  • Learn from other developers

Action Shortcut

Knowledge Check

How do you inspect an element?

  • Right-click → Inspect
  • Press F12
  • Use the Select Element tool

What does a strikethrough CSS rule mean?

The CSS property has been overridden and is not currently applied.

Can you edit CSS in DevTools?

Yes. Changes appear immediately, but they are temporary and disappear after refreshing the page.

Quick Summary

  • DevTools is used for debugging websites.
  • Elements panel shows HTML.
  • Styles panel shows CSS.
  • Open DevTools using F12 or Inspect.
  • Use the Select Element tool to inspect webpage elements.
  • Strikethrough CSS means the rule has been overridden.
  • CSS edits in DevTools are temporary.
